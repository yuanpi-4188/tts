# tts
一个命令行 TTS 工具。能说话，五种声音，语速音量可调，支持保存。装完就能用。
# 安装
运行 tts_setup.exe 即可。

# 用法
tts -xx 你好
tts -xy -r+30 慢慢说
tts -yx -v+10 -s=out 保存声音
tts -yj -r-20 -s=./data/1 从文件读.txt

# 音色
-xx 晓晓（女）
-xy 晓伊（女）
-yj 云健（男）
-yx 云希（男）
-ya 云扬（男）

# 参数
-r+30 语速 +30%
-r-20 语速 -20%
-v+10 音量 +10%
-v-20 音量 -20%
-s=文件名 保存为 MP3（自动加后缀）

# 说明
支持从文件读取：如果输入的是存在的文件路径，自动读文件内容
保存文件自动补 .mp3，不用自己加
不保存时自动播放，播完删临时文件
